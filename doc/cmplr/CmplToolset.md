## cmpl Toolset

the cmpl toolset is supposed to be an intuitive way to create a compiler for each platform. Any functions will be described [here](../../asm/cmpl/cmplToolset.h)

The toolset can be used to create your own toolsets for each compiler. For example, The OS Toolset, which is the most basic and can be seen [here](../../asm/cmpl/os/funcToolset.h)
