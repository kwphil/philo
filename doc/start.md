# Philo
If you want to learn how to use Philo, This is a great place to start.

First off, all programs will start with:
```
fn public int main() {
```
or:
```
fn public int main(int argc, string argl[]) {
```
The second line accepts the arguments from the command line into the program. The first should only be used with either basic programs or bootloaders. Programs without either of these will return errcode: ```0x077```
