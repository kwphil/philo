## Phacket

Phacket is an extremely simple package manager, and pulls its packages off of github [here](https://www.github.com/kwphil/phacket/main)

You can call the package manager with the command phacket(note that this currently only runs off of linux). If you want a list of packages to install call 
```
Phacket LIST
```

If you want to compile a package, call
```
Phacket <package>
```
